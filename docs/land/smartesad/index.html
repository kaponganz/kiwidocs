Чому не PWM?

✘ PWM

  • ✘ 2 окремих дроти (ARM + FIRE)
  • ✘ Одностороннє керування — пілот не знає стан пристрою
  • ✘ Немає контролю помилок — збій = мовчання
  • ✘ Фіксована логіка, без налаштувань

✔ SmartESAD

  • ✔ UART (full-duplex або half-duplex)
  • ✔ Двосторонній зв'язок — статус завжди на OSD
  • ✔ Checksum на кожному повідомленні — помилка = попередження
  • ✔ Настроювані таймери, пороги, затримки
