Чому H743-Wing?

✘ Звичайний FC

  • ✘ Один IMU — збій сенсора = втрата контролю
  • ✘ Обмежене живлення 4–6S
  • ✘ Мало PWM виходів — не вистачає для крила
  • ✘ Без OSD або зовнішній MAX7456

✔ H743-Wing

  • ✔ Подвійний IMU з апаратною фільтрацією сигналу
  • ✔ 12S живлення + 3 незалежних BEC
  • ✔ 15 PWM виходів — 8 моторів + 7 серво
  • ✔ OSD на STM32 + MSP Displayport для цифрових VTX

Можливості

Все для автономних місій на великі відстані.

IMU×2

Подвійний IMU

ICM-42688P + ICM-45686 з апаратною фільтрацією. Резервування на рівні заліза.

12S

12S живлення

3 незалежних BEC: 5V, 6/7V, 9/12V — кожний до 5A.

15

15 PWM виходів

8 моторів + 7 серво. Повний контроль крила, VTOL та складних конфігурацій.

OSD

OSD на STM32

Аналогове OSD без MAX7456. MSP Displayport для DJI O3, HDZero, Walksnail.

GPS∅

GPS-less зліт

Автономний запуск без GPS-фіксу. Для катапульти та ручного старту.

8+CAN

8 UART + CAN

Повна підтримка периферії: GPS, телеметрія, далекомір, OSD, CAN-пристрої.

Плати

Flight Controller + Power Distribution Board.
